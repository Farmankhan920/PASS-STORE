Pass Store
Secure Your Digital Identity
v2.0.0 Stable

Pass Store is a modern, secure password manager built with a liquid glass aesthetic inspired by next-generation iOS design. All your passwords are encrypted and stored per-user in Firebase Firestore.

Features
🌐
Tri-lingual
Full support for English, Hindi and Hinglish
🔐
Per-User Vault
Each account has its own isolated password store in Firestore
✨
Liquid Glass UI
iPhone 17 inspired blur, frosted glass design system
🛡️
2FA Support
Store your two-factor authentication contacts alongside passwords
🔑
Password Generator
Generate cryptographically secure passwords instantly
📱
APK Ready
Mobile-first design optimized for HopWeb APK conversion
